## 🧠 Qué es una SKILL en este proyecto

En esta serie, una **skill** es un archivo markdown que define con claridad:

- Qué tarea específica hace Claude por ti.  
- Cuándo debe activarse.  
- Qué pasos debe seguir.  
- Qué formato debe tener la respuesta.

Tú no tienes que “programar”: solo copias el contenido del archivo `.md`, lo pegas en Claude (o lo usas dentro de tu workspace) y dejas que la skill haga el trabajo.

## 📨 Skill Emails · Responder correos de Gmail con tu tono y estilo de comunicación

**Nombre de la skill:** `responder-emails-gmail`  
**Repositorio:** `skill-emails`  
**Archivo de la skill:** `skill-responder-emails-gmail.md`  

**¿Para qué sirve?**  
Para que Claude prepare borradores de respuesta en Gmail con tu tono y estilo de comunicación, sin enviar nada automáticamente.

**Cuándo usarla:**

- Cuando quieras responder un correo real que está en tu Gmail.  
- Cuando menciones remitente, asunto o el contenido del email y pidas algo como “déjame el borrador listo”.

**Qué hace paso a paso:**

1. Identifica el correo correcto usando los datos que le das.  
2. Lee el hilo para entender el contexto.  
3. Redacta una respuesta clara, cercana y profesional con tu estilo.  
4. Crea un borrador en Gmail (si hay integración) o te devuelve el texto para copiar y pegar.

## 🚀 Cómo usar una skill desde cero

1. Abre el archivo de la skill que te interesa (por ejemplo, Skill Emails).  
2. Lee “Qué hace” y “Cuándo usarla” para confirmar que resuelve tu caso.  
3. Abre Claude (web, desktop o Claude Code).  
4. Copia el contenido de la skill o cárgala como archivo en tu workspace.  
5. Dale a Claude el contexto real (correo, presentación, documento, etc.) y deja que la skill haga el trabajo.
